# Как открыть Windows PowerShell на Windows

Windows PowerShell — это окно, куда вы будете вводить команды.
Через него мы будем:

* устанавливать и запускать базу данных **PostgreSQL**,

Дальше — по шагам.

---

## Шаг 1. Открываем нужный Windows PowerShell

1. Нажмите кнопку **«Пуск»**.
2. Пролистайте список программ до папки **«Windows PowerShell»** и откройте её.
   (Если лень листать — можно в поиске Пуска начать печатать `PowerShell`.)
3. В этой папке вы увидите несколько вариантов:

   * без пометки — обычный PowerShell,
   * с подписью `x86` — 32-битная версия.

Ориентируйтесь на разрядность вашей системы (**x86 или x64**), которую вы уже узнавали в гайде
[how_install_pycharm.md](how_install_pycharm.md).
Обычно для **Windows x64** достаточно обычного **Windows PowerShell** без `x86` в названии.

Главное: **открывайте один и тот же вариант каждый раз**, чтобы не путаться.

---

## Шаг 2. Запускаем терминал от имени администратора

1. Найдите нужный вариант **Windows PowerShell** (из предыдущего шага).
2. Нажмите по нему **правой кнопкой мыши**.
3. Выберите пункт **«Запуск от имени администратора»**.
4. Если появится окно «Разрешить этому приложению вносить изменения?» — нажмите **«Да»**.

Откроется тёмно-синее или чёрное окно с мигающим курсором справа от строки вида:

```text
PS C:\Users\ВашеИмя>
```

Это и есть ваше «рабочее место» для команд.

## Если что-то пошло не так

Если вы:

* не нашли нужную кнопку,
* не нашли папку **Windows PowerShell**,
* увидели страшную ошибку,
* запутались, какое именно приложение открывать,

**не паникуйте**. Скопируйте:

1. Текст шага из этого гайда, где вы застряли.
2. Текст ошибки или сделайте скриншот и перепишите, что там написано.
3. Коротко опишите, **что вы делали до этого момента**
   (а лучше заранее записывайте все ваши шаги по установке).

И напишите любому ИИ-ассистенту (ChatGPT, Claude, Grok, DeepSeek и т.п.) примерно так:

> *«Я впервые пытаюсь найти и запустить Windows PowerShell по этому гайду. Я застрял на шаге [номер шага]. Вот текст шага и вот ошибка. Вот что я уже сделал до этой ошибки. Помоги, пожалуйста, разобраться, что мне сделать дальше».*

Такие ассистенты отлично умеют объяснять подобные вещи простым языком и подсказывать следующий шаг.
